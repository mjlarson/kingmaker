# Installation

## Requirements

- Python 3.12+
- `numpy`
- `scipy`
- `numba`
- `matplotlib`
- `healpy`

## From source

```bash
git clone https://github.com/mjlarson/kingmaker.git
pip install ./kingmaker
```

## Development install

To install with testing and linting tools:

```bash
pip install -e "./kingmaker[dev]"
```
